## Hello from `Kokoserver`

Nothing like a quick API to say hi.

```bash
pip install fastapi uvicorn
```

Then, inside `app.py`:

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def welcome():
    return {"message": "Welcome to Kokoserver's Github"}

@app.get("/{user}")
def user(user: str):
    return {"message": f"Welcome to Kokoserver's Github, {user}"}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, port=8000)
```

Run it with `uvicorn app:app --reload` and hit the endpoints. Have fun!
